# @sieutoc/themes

A theme collection for Sieutoc site generator

## Developing

Clone the project

```bash
git clone git@github.com:websitesieutoc/themes.git
```

Go to the project directory

```bash
cd themes
```

Install dependencies

```bash
pnpm install
```

Start the server

```bash
pnpm dev
```

Copy the based theme

```bash
cp -r src/minimalist src/newname
```
